Installation

Install GlocalText using pip. Requires Python 3.10+.

pip install glocaltext

Quick Start

Get up and running in seconds.

  1. Initialize your project:

    glocaltext init .

  2. Edit the generated config file at .ogos/glocaltext/configs/main.yaml.

  3. Run the translation:

    glocaltext run .

glocaltext init

Initializes a new GlocalText project in the specified directory.

glocaltext init [PATH]
  • Creates the .ogos directory structure.
  • Generates a default main.yaml configuration file.

glocaltext run

Executes the translation tasks defined in your configuration.

glocaltext run [PATH]
  • [PATH]: The root directory of your project (containing .ogos).

Flags & Options

Common flags you can use with glocaltext run:

  • --dry-run: Simulate the process without making changes or calling APIs.
  • --incremental: Only translate new or modified files (overrides config).
  • --debug: Enable verbose logging for troubleshooting.
  • --version: Show the installed version.

Providers

Configure your AI translation providers in main.yaml.

providers:
  gemini:
    api_key: 'YOUR_KEY'
    model: 'gemini-2.5-flash-lite'
    rpm: 60
  google: {}  # Uses Google Translate (no key required)

Tasks

Define what to translate. Each task specifies source files, target language, and output location.

tasks:
  - name: 'Translate Docs'
    target_lang: 'ja'
    source:
      include: ['docs/**/*.md']
    output:
      path: 'docs/ja'

Rules (Firewall)

The core of GlocalText. Use regex to control exactly what gets translated.

  • Protect: Prevent specific patterns (like variables) from being translated.
  • Skip: Skip entire lines or blocks.
  • Replace: Pre-process text before translation.
rules:
  protect:
    - '\$VAR'       # Protect $VAR
    - '`[^`]+`'     # Protect code in backticks
  replace:
    'User: (.*)': '使用者: \1'
